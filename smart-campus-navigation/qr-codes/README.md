# Campus QR Code Examples

Use these links to generate QR codes for campus buildings. Each QR code should point to the deployed frontend with the `start` query parameter already included.

## Example smart links

| Building | QR target link |
| --- | --- |
| Gate | `https://smart-campus-navigation.vercel.app/?start=Gate` |
| Admin | `https://smart-campus-navigation.vercel.app/?start=Admin` |
| Main Building | `https://smart-campus-navigation.vercel.app/?start=Main%20Building` |
| Lobby | `https://smart-campus-navigation.vercel.app/?start=Lobby` |
| Library | `https://smart-campus-navigation.vercel.app/?start=Library` |
| Lab A | `https://smart-campus-navigation.vercel.app/?start=Lab%20A` |
| Canteen | `https://smart-campus-navigation.vercel.app/?start=Canteen` |
| Auditorium | `https://smart-campus-navigation.vercel.app/?start=Auditorium` |

## How to generate the QR images

1. Open any QR code generator such as [QR Code Monkey](https://www.qrcode-monkey.com/) or [QRStuff](https://www.qrstuff.com/).
2. Paste one of the smart links from the table above.
3. Generate the QR code.
4. Download it as PNG or SVG.
5. Print and place the QR code near the corresponding building entrance.
